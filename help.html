Command Palette

An extension that allows you to control Firefox with a Sublime Text/helm-M-x inspired user interface.

How do I use it?

  1. Activate the command pallete. You can do this three ways:
    1. Focus the omnibar (e.g. with ctrl+l) then type ;  (that's with a space at the end). The top suggestion will be a message confirming the command palete is active
    2. Click the button to open a popup with the command palette inside.
    3. Press Ctrl+Space to open a popup underneath the button (as if you clicked on it).
  2. Type in words related to the task you want to complete. For example, to find all commands that affect tabs, type in tab  and see what suggestions it gives.

    Note: the omnibar can only show 6 suggestions at a time, so this isn't an exhaustive list. The page you are currently reading lists all available commands.

  3. Once you can see the command you want, use tab key to select it.
  4. optional If you're chosen command takes an argument, suggestions will now be listed. Use the familar search then tab interface to select the argument you want. Free-form arguments are not allowed.
  5. Invoke the command with return.
In the popup interface, the mouse can also be used. Clicking once on an item will fill your input, whilst double-clicking will activate the command.

Commands

help
Open the help page (what you're currently reading) in a new tab.
open new tab
Open a new tab.
open tab in container container-id
Open a new tab in the specific container. Auto-complete will help you find the right ID.
open tab in temporary container
Open a new tab in a temporary container (requires the 'Temporary Containers' extension).
reload this tab
Reload this tab.
put this tab to sleep
Unload this tab and move focus to a random, non-sleepy tab.
put other tabs to sleep
Finds all other tabs in the current window (that aren't pinned), and puts them to sleep. The current tab will stay awake.
mute this tab
Mute audio from this tab.
unmute this tab
Unmute audio from this tab.
pin this tab
Pin this tab.
unpin this tab
Unpin this tab.
close this tab
Close this tab.
open new window
Open a new window.
open new private window
Open a new private window.
close this window
Close this window.
toggle reader mode in this tab
Toggle reader mode in this tab.
save this tab as pdf
Save this tab as pdf.
move this tab to container container-id
Close this tab then open its URL in the chosen container. Takes the container ID as a required argument. The container name each ID maps to is given in the description.
move this tab to temporary container
Close this tab then open its URL in a new, temporary container (requires 'Temporary Containers' extension).
go to tab tab-id
Change focus to the given tab. Takes the target tabs ID as a required argument (which autocomplete should help you find).
move this tab to popup window
Move the current tab to its own window with a minimal user interface. There is no way of reversing this operation that I'm aware of.
move this tab to a new window
Move the current tab to its own window.

Why the semicolon?

I use ctrl+l to focus the omnibar on Firefox, and ; is next to l on my keyboard. So ; is the most convenient key for me to press after focusing the omnibar.
